Таблиці:
    Переваги:
        Простота розмітки: Таблиці можуть бути простим способом організації вмісту, особливо якщо потрібно відобразити дані у вигляді табличної структури.
        Стабільність: Елементи у таблиці розміщаються відносно один одного, що може забезпечувати стабільніше позиціонування.
    Недоліки:
        Непридатність для адаптивного дизайну: Використання таблиць може ускладнити створення адаптивного дизайну для мобільних пристроїв та інших розмірів екранів.
        Семантична недоречність: Використання таблиць для розмітки макету може бути несемантичним, 
        оскільки таблиці призначені для представлення табличних даних, а не для розмітки структури документа.
Блоки:
    Переваги:
        Гнучкість: Блоки дають більшу гнучкість у розміщенні елементів на сторінці, що дозволяє створювати складніші макети.
        Адаптивність: Використання блоків спрощує створення адаптивного дизайну, оскільки дозволяє легко адаптувати розміщення елементів для різних пристроїв.
    Недоліки:
        Складність: Розміщення блоків може бути складнішим, особливо для новачків, оскільки вимагає більшої кількості CSS-коду для досягнення бажаного результата.
        Залежність від контенту: При використанні плаваючих блоків можуть виникати проблеми з розміщенням випливаючих блоків, якщо контент на сторінці змінюється.

JavaScript може бути застосований на багатьох рівнях та в різних сферах розробки. Деякі з них включають веб-розробку (фронтенд та бекенд), розробку додатків, роботу з базами даних, розробку ігор, автоматизацію завдань та інше.

URL (Uniform Resource Locator) - це адреса ресурсу в Інтернеті. Схема URL вказує на протокол, за допомогою якого браузер або інший клієнт отримує доступ до ресурсу.

Обробники подій в JavaScript використовуються для виконання певних дій, коли відбувається певна подія, така як клік мишею, натискання клавіші на клавіатурі, завантаження сторінки тощо. Вони дозволяють створювати інтерактивні веб-сторінки та додавати реакцію на дії користувачів.

Підстановка (в контексті програмування) означає заміну певних значень у рядку, об'єкті або шаблоні на інші значення. Це може бути корисним при форматуванні рядків або шаблонів, а також при виведенні даних на сторінку або у відповідь на запит.

Вставка скриптових тегів script у HTML-сторінку дозволяє виконувати JavaScript-код прямо у контексті сторінки. Це може бути корисним для додавання динамічного функціоналу, обробки подій та взаємодії з користувачем. Тег script можна використовувати як внутрішній (вставлений безпосередньо в HTML-документ) так і зовнішній (вказано шлях до зовнішнього JavaScript-файлу).
